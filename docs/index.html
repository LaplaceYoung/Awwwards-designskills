最佳实践提示词

1.0 样本提示：武汉市旅游宣传页

请使用 awwwards-design-selector 技能生成“武汉市旅游宣传页面 1.0”。
要求：
1) 中文优先，包含 Hero、亮点卡片、一日路线、咨询表单；
2) 至少 3 张武汉旅游相关图片（黄鹤楼/东湖/长江夜景）；
3) 首次交付前必须跑 smoke test 并输出截图。
（零样本默认：复刻本年度高分网站设计语言）
输出目录：output/awwwards-design-selector/wuhan-tourism-v1/

2. 迭代提示：武汉大学宣传页增强

请基于已有武汉大学宣传页继续精进，并保持参考站动效语言。
任务：
1) 强化章节滚动叙事与点击反馈；
2) 替换业务相关媒体素材；
3) 输出 before/after 对比截图；
4) 提供 fidelity 与 smoke 验证结果。
输出目录：output/awwwards-design-selector/whu-promo-gq-hifi/
